# 📚 Zai Education Hub

**Free Quality Education for Every Sri Lankan Student 🇱🇰**

Zai Education Hub is a free educational platform providing study materials, class recordings, GK quizzes, educational games, and more — covering the entire Sri Lankan curriculum from Grade 1 to A/L.

## ✨ Features

- **📚 Study Materials** - Subject-wise resources for AL, OL, Grade 6-9, and Grade 1-5
- **🎥 Class Recordings** - Video lessons from expert teachers
- **🧠 GK Quiz** - Real-time general knowledge quizzes with timer & scoring
- **❓ Doubt Clearing** - Connect via WhatsApp, Email, Telegram, or Phone
- **🤖 Bots** - WhatsApp & Telegram bots for real-time updates
- **🎮 Educational Games** - Memory Match, Word Puzzle, Math Speed & more
- **🔗 Useful Links** - Curated external resources for deeper learning

## 🚀 Quick Start

Simply open `index.html` in your browser, or visit [zaieduhub.github.io](https://zaieduhub.github.io/).

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties, Animations, Flexbox, Grid)
- Vanilla JavaScript (IntersectionObserver, no frameworks)
- Google Fonts (Baloo 2 + Comic Neue)
